# «Лабиринт сокровищ» — консольная игра

Проект для учебного задания: консольная игра-лабиринт с комнатами, предметами, загадками, ловушками и случайными событиями.

## Требования

- Python **3.11+**
- Poetry

## Установка

```bash
make install
```

## Запуск игры

```bash
make project
```

## Команды

- `help` — справка
- `look` — осмотр комнаты
- `go <dir>` — перемещение (`north`, `south`, `east`, `west`)
- `take <item>` — взять предмет
- `use <item>` — использовать предмет
- `solve` — решить загадку (если есть)
- `inventory` — инвентарь
- `status` — здоровье/статус
- `quit` — выход

## Линтер и сборка

```bash
make lint
make build
```

## Победа и завершение игры

Победа достигается, когда игрок **забрал `treasure`** и **вернулся в комнату входа**.
К сокровищу можно попасть двумя способами: **использовать ключ (`key`)**, либо **решить загадку (`solve`) и открыть проход руной (`rune`)**.
Игра корректно завершается командой `quit` или при падении HP до 0.
