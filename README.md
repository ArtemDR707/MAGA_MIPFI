# MAGA_MIPFI — два проекта

В этом архиве лежат два независимых Poetry-проекта:

1) `labyrinth_treasures/` — игра «Лабиринт сокровищ»
2) `primitive_db/` — проект «Примитивная база данных»

## Быстрый старт

### Лабиринт
```bash
cd labyrinth_treasures
make install
make lint
make project
```

### Primitive DB
```bash
cd primitive_db
make install
make lint
make project
```
